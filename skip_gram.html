    Skip-gram

    Skip-gram sampling

    Skip-grams is a technique where by n-grams are still stored to model language, but they allow for tokens to be skipped. This method can overcome the data sparsity problem to some extend1.

    Skip-gram can sample the context in the way of "jumping". For example, set a k of n-gram skip-gram model. k means the range of word-skip and n is still the concept of n-gram. Give out a sentence of w1 to w10. If we set the current word is w5. In this case, we use a model of 3 of 2 skip-gram. Then, the context would be extract as 2-gram and the word-skip be 4. the prefix context is {w2,w3,w4} and the surfix context is {w6,w7,w8}. The model would sample 2 words from context either. Sampling result would be {wx,wy|x,y from (w2,w3,w4,w6,w7,w8)}
